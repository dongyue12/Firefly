### 一、常用快捷键速查表

| 功能               | 快捷键 (Windows/Linux)         |
| ------------------ | ------------------------------ |
| 快速打开最近的文件 | `Ctrl + P`                     |
| 查找（当前文件）   | `Ctrl + F`                     |
| 替换（当前文件）   | `Ctrl + H`                     |
| 全局查找           | `Ctrl + Shift + F`             |
| 全局替换           | `Ctrl + Shift + H`             |
| 粘贴并格式化       | `Ctrl + Shift + V`             |
| 复制               | `Ctrl + C`                     |
| 撤销               | `Ctrl + Z`                     |
| 打开命令面板       | `Ctrl + Shift + P`             |
| 打开设置           | `Ctrl + ,`                     |
| 向左缩进           | `Ctrl + [`                     |
| 向右缩进           | `Ctrl + ]`                     |
| 单行注释           | `Ctrl + /`                     |
| 多行注释           | `Ctrl + Shift + /`（可自定义） |
| 格式化代码         | `Shift + Alt + F`              |

### 二、常用操作技巧

#### 1. 快速复制一行

- **方式一**：`Shift + Alt + ↑`（向上复制） / `↓`（向下复制）
- **方式二**：`Ctrl + C` 复制当前行，`Ctrl + V` 粘贴

#### 2. 选定多个相同单词（批量修改）

- 双击选中一个单词，然后按 `Ctrl + D`
- 每按一次会继续向下选中下一个相同单词，可同时编辑所有选中项

#### 3. 添加多个光标

- `Ctrl + Alt + ↑`（向上添加光标）/ `↓`（向下添加光标）
- 适合在多个位置同时输入相同内容

#### 4. 全局替换某个单词

- 按 `Ctrl + H` 打开替换面板
- 输入查找内容和替换内容后，选择**全部替换**（替换所有匹配项）

#### 5. 快速定位到某一行

- 按 `Ctrl + G`，输入行号后回车，直接跳转

#### 6. 选择某个区块（列选择/块选择）

- 按住 `Shift + Alt`，然后用鼠标拖动选择矩形区域

#### 7. 放大 / 缩小整个编辑器界面

- 放大：`Ctrl + +`（或 `Ctrl + =`）
- 缩小：`Ctrl + -`

### 三、自定义快捷键示例

VSCode 支持自定义快捷键，避免与默认快捷键冲突，建议先熟悉默认设置后再修改。

**操作步骤：**

1. 按 `Ctrl + K`，然后按 `Ctrl + S`，打开键盘快捷方式设置（macOS：`⌘K ⌘S`）
2. 在搜索框中输入命令名称，如 **"切换块注释"**
3. 找到对应条目，点击左侧的**编辑**按钮
4. 直接按下你想要的新快捷键组合，如 `Ctrl + Shift + /`
5. 按回车保存即可

> 提示：多数快捷键均可通过此方法自定义，修改前建议确认不与系统或其他命令冲突。
